== 개요

HR 관리 시스템은 인사 관리를 위한 마이크로서비스 아키텍처 기반의 애플리케이션입니다.

=== 시스템 아키텍처

이 시스템은 다음과 같은 특징을 가지고 있습니다:

* Spring Boot 3.5.5 기반
* Java 17 사용
* Spring Security 보안 적용
* Spring Data JPA를 통한 데이터 관리
* H2 Database (개발 환경)
* Spring REST Docs를 통한 API 문서화

=== API 공통 규약

==== HTTP 상태 코드
* `200 OK` - 성공
* `400 Bad Request` - 잘못된 요청
* `401 Unauthorized` - 인증 실패
* `403 Forbidden` - 권한 없음
* `404 Not Found` - 리소스 없음
* `500 Internal Server Error` - 서버 오류
* `503 Service Unavailable` - 서비스 사용 불가

==== 응답 형식
모든 API는 JSON 형식으로 응답합니다.
