----
{
  "status": 404,
  "error": "Employee not found with sabun: 99999",
  "timestamp": "2025-09-22T10:30:00"
}
----

==== 유효성 검증 규칙

[cols="1,1,3"]
|===
|필드명 |필수여부 |규칙

|`sabun`
|Y
|사번은 필수이며, 중복될 수 없습니다

|`name`
|Y
|이름은 필수입니다

|`birYmd`
|N
|생년월일 (YYYYMMDD 형식)

|`empYmd`
|N
|입사일 (YYYYMMDD 형식)

|`email`
|N
|이메일 형식이어야 합니다

|`address`
|N
|주소 정보
|===

[source,json]
----

  "timestamp": "2025-09-22T10:30:00"
}
----
